# rp2350

Clignotement d'une LED pour le RP2350.

## Fichiers du projet

- `blink.c` : Code source C du firmware
- `CMakeLists.txt` : Configuration CMake
- `build.sh` : Script de compilation
- `GUIDE.md` : Guide d'utilisation

## Objectif

Créer un firmware UF2 pour clignoter une LED, avec la même organisation que le projet RP2040.
